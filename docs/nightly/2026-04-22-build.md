### Acceptance criteria

- Matrix has three entries: desktop-chromium, mobile-iphone, mobile-pixel.
- Each shard installs only the browser(s) it needs (chromium for
  desktop + pixel, webkit for iphone) and the cache key includes the
  browser name so shards don't clobber each other.
- Each shard runs `npx playwright test --project=<project>`.
- Per-shard `timeout-minutes` set to 15 to enforce the budget.
- `fail-fast: false` so one shard failing doesn't hide the others.
- Per-shard report artifacts (`playwright-report-${shard}`).
- `npm run typecheck` and `npm test` still pass (no source changes, so
  this is a sanity check).
